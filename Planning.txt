Use Case
~~~~~~~~

Debug:
	render the world

User:
	Create a physics world
	Add global forceConstraint
	Add RigidBody
	Tick the world
	Add a collision callback
	Get the status of an object
	Add a collision filter
	
Engine:
	generate contact points
	solve contact points
	apply forces
